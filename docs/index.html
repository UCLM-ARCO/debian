ARCO Debian package repository

To use this package repository, execute the commands bellow: 
      $ sudo su
      # echo "deb http://pike.esi.uclm.es/arco sid main" > /etc/apt/sources.list.d/pike.list
      # apt-get update
      # apt-get install arco-archive-keyring
or just: 
      $ wget -qO- http://pike.esi.uclm.es/add-pike-repo.sh | sudo sh
To import apt key only: 
      $ sudo apt-key adv --fetch-keys http://pike.esi.uclm.es/arco/key.asc
or 
      $ sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys D917ABDD28380433
