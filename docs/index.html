ARCO Debian package repository

To use this package repository: 
$ curl -s  https://uclm-arco.github.io/debian/add-repo.sh | sudo bash
Or by hand: 
$ sudo wget -P /etc/apt/trusted.gpg.d https://uclm-arco.github.io/debian/uclm-arco.asc
Alternatives: 
$ wget -qO- https://uclm-arco.github.io/debian/uclm-arco.asc | sudo tee /etc/apt/trusted.gpg.d/uclm-arco.gpg
$ gpg --keyserver keyserver.ubuntu.com --recv-key D917ABDD28380433; gpg --export D917ABDD28380433 | sudo cat > /etc/apt/trusted.gpg.d/uclm-arco.gpg
Deprecated methods: 
$ sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys D917ABDD28380433
$ sudo apt-key adv --fetch-keys https://uclm-arco.github.io/debian/uclm-arco.asc
And add the repository: 
$ echo "deb https://uclm-arco.github.io/debian/ sid main" | sudo tee /etc/apt/sources.list.d/arco.list
$ sudo apt update
